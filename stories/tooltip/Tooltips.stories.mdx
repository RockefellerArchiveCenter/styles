# Tooltips

Temporarily display help or confirmation messages associated with a button or 
other action element.

## When to use

Use the tooltip component when you want to temporarily display a brief help or 
confirmation message that is associated with a clickable element.

## Best practices

- Tooltips should be displayed for at least six seconds.
- Use the role='alert' attribute to support screen readers.
